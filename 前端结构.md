# 📜 一、项目目标

构建一个 **Vue 3 + Vite** 的前端项目，包含两部分：

1. **用户端：**简单的反馈提交页面（文本 + 联系方式）
2. **后台管理端：**登录页 + 反馈列表页 + 分页功能

---

你可以直接把下面的结构贴给 Cursor：

```
frontend/
```

# 🔍 三、主要页面说明

---

## 1️⃣ 用户反馈页：FeedbackForm.vue

包含：

* 文本输入框（反馈内容）
* 联系方式输入框
* 提交按钮
* 调后端：POST /api/feedback/add

---

## 2️⃣ 后台登录页：AdminLogin.vue

功能：

* Administrator 登录
* 登录成功保存 token（localStorage 或 pinia）
* 跳转到后台列表页

调用接口：

```
POST /admin/login
```

---

## 3️⃣ 后台列表页：FeedbackList.vue

功能：

* 分页请求
* 搜索
* 表格展示
* 时间排序
* 使用 token 来访问接口

接口示例：

```
GET /admin/feedback/list?page=1&size=10&keyword=
```

# 🧱 四、核心代码示例（可直接复制给 Cursor）

下面是你可以让 Cursor **自动创建文件** 的内容示例。

---

# 📌 main.js

```javascript
import { createApp } from 'vue'
import App from './App.vue'
import router from './router'
import { createPinia } from 'pinia'

createApp(App)
  .use(createPinia())
  .use(router)
  .mount('#app')
```

---

# 📌 router/index.js

```javascript
import { createRouter, createWebHistory } from 'vue-router'
import FeedbackForm from '../pages/FeedbackForm.vue'
import AdminLogin from '../pages/AdminLogin.vue'
import FeedbackList from '../pages/FeedbackList.vue'

const routes = [
  { path: '/', component: FeedbackForm },
  { path: '/admin/login', component: AdminLogin },
  { path: '/admin/feedback', component: FeedbackList }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

export default router
```

---

# 📌 api/request.js（axios 封装）

```javascript
import axios from 'axios'

const instance = axios.create({
  baseURL: 'http://localhost:5000',
  timeout: 5000
})

instance.interceptors.request.use(config => {
  const token = localStorage.getItem('token')
  if (token) config.headers.Authorization = `Bearer ${token}`
  return config
})

export default instance
```

---

# 📌 api/feedback.js

```javascript
import request from './request'

// 用户提交反馈
export function submitFeedback(data) {
  return request.post('/api/feedback/add', data)
}

// 管理员登录
export function adminLogin(data) {
  return request.post('/admin/login', data)
}

// 后台分页查询
export function fetchFeedbackList(params) {
  return request.get('/admin/feedback/list', { params })
}
```

# ✅ 六、核心结论

你的前端可以非常轻量，最实用的结构是：

> **Vue 3 + Vite + Axios + Pinia + Vue Router**
>
> 页面包含：FeedbackForm（用户） + AdminLogin + FeedbackList（后台）
